# Recommendation-System
A rule-based AI recommendation system built with Python that suggests movies based on user preferences using pattern matching and recommendation logic.

## Features

* User preference input
* Multiple interest selection
* Similarity score calculation
* Personalized recommendations
* Ranking of recommended movies
* Interactive command-line interface

## Technologies Used

* Python
* Dictionaries
* Lists
* Loops
* Conditional Logic
* Similarity Matching

## How It Works

1. The user enters one or more interests.
2. The system compares those interests against movie tags.
3. A similarity score is calculated for each movie.
4. Movies are ranked based on their scores.
5. The top matching recommendations are displayed.

## Example

### User Input

```text
action, sci-fi
```

### Recommendations

```text
⭐ The Matrix | Match Score: 2
⭐ Inception | Match Score: 2
⭐ John Wick | Match Score: 1
⭐ The Dark Knight | Match Score: 1
⭐ Free Guy | Match Score: 1
```
